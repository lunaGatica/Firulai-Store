# Firulai-Store
Tienda de mascotas: "Encontra todo lo que tu peludo necesita"
Tecnologias utilizadas:
*React 18
*Vite
*React Router DOM v6
*CSS Modules
*Context API
*Fetch API
El carrito utiliza Context API.
*Agregar productos
*Eliminar productos
*Vaciar carrito
*Compartir datos entre componentes
Funcionalidades:
 *Layout con Header y Footer
 *Navegación con React Router
 *Catálogo dinámico desde JSON
 *Componentes reutilizables
 *Vista detalle de productos
 *Diseño responsive básico
